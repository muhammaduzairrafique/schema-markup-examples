# Schema Markup Examples

A collection of simple JSON-LD schema markup examples for SEO and structured data implementation.

## Schema Types Included

- Organization Schema
- LocalBusiness Schema
- Service Schema
- FAQPage Schema
- BreadcrumbList Schema
- Article Schema
- Website Schema

## Purpose

Schema markup helps search engines better understand website content, business details, services, FAQs, and page relationships.

## Note

These examples should be customized based on the real business, website, service, page, and location details before implementation.
